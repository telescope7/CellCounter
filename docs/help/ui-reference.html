UI Button and Control Reference

Detailed description of each control in the main Cell Counter window.

Top-Right Header Controls

ControlWhat It Does
Help Opens this local HTML documentation set in the system browser.
Status Chip Shows pipeline state. Common values: Idle, Loaded, Playing, Paused, Seeking, Fast Analyze, Configured, Complete, Error.

Controls Card: Primary Row

ButtonBehavior
Open VideoOpen a file chooser and initialize video + analysis pipeline.
Fast AnalyzeRun accelerated full-pass analysis without realtime playback timing.
Play/AnalyzeToggle realtime play/pause while tracking each frame.
StepAdvance exactly one frame while paused.
ReplayReset analysis and frame pointer to frame 1 using current settings.
Save ResultsSave both analysis and footprint CSVs in one action.
SimulatorOpen Cell Simulation Studio to generate synthetic videos and ground truth files.

Controls Card: Secondary Row

ControlBehavior
Frame label (Frame: X/Y) Displays current frame index and total frame count.
Video position slider Seek to a specific frame. During seek, controls are disabled until render completes.
Playback Speed label (Playback Speed: Nx) Displays active playback rate for realtime mode.
Playback speed slider Adjust play rate (0.1x to 5.0x).
Tune Detection Open live detection tuning dialog for segmentation/tracking parameters.
Mask View Toggle MOG2 foreground mask display in the video pane.

Visualization Panels

PanelWhat You See
Video panelCurrent processed frame, overlays, and optional mask view.
Track Start DistributionHistogram + cumulative distribution of track-start times.
Speed DistributionHistogram + cumulative distribution of observed speeds.

Tune Detection Dialog Controls

Dialog ControlBehavior
Parameter slidersAdjust segmentation/tracking parameters and trigger preview when released.
Enable MOG2 shadowsToggle shadow classification in background subtraction.
Preview mask viewPreview tuning against mask output instead of composited frame.
Apply ParametersApply current dialog values to active runtime configuration.
Reset SlidersRestore sliders to currently applied configuration.
CloseClose dialog and return to main window.

When preview rendering is in progress, slider and parameter input controls are temporarily disabled to prevent race conditions.

For exact parameter behavior, see Configuration Reference.
