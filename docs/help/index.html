Quick Start (GUI)

  1. Click Open Video and choose a source file.
  2. Set frame location and playback speed as needed.
  3. Use Tune Detection on a representative paused frame.
  4. Run with Play/Analyze or Fast Analyze.
  5. Save outputs with Save Results.

Quick Start (Headless)

java -jar cellcounter.jar /path/video.avi /tmp/run01 \
  --tracking-config=/path/CellCounter.properties

Headless mode uses the same tracking engine and config semantics as GUI mode.
